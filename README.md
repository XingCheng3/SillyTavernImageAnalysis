# SillyTavern Image Analysis

一个偏前端本地使用的 **SillyTavern 角色卡编辑 / 世界书编辑 / AI 翻译 / PNG 导出工具**。

## 当前定位

这个项目当前更适合：

- 编辑 **PNG 角色卡**
- 解析并编辑角色卡内的基础信息 / 高级信息 / 世界书
- 批量 AI 翻译并做对比后选择性应用
- 将修改后的角色卡重新导出为 **PNG**
- 导入 **JSON 角色卡** 作为编辑入口（导出仍以 PNG 为主）

这个项目**不是**完整的在线 SaaS，也**不是**通用资产管理平台。当前形态更接近：

> 面向个人 / 小团队的 SillyTavern 角色卡与世界书编辑工具

## 主要能力

- PNG 角色卡导入与解析
- JSON 角色卡导入
- 角色基础信息编辑
- 高级字段编辑
- 世界书条目编辑
- 基础 / 高级 / 世界书批量翻译
- 翻译结果对比后应用
- 全局替换
- 快照 / 撤销 / 恢复初始
- PNG 角色卡导出

## 目前重点兼容方向

- `chara`
- `ccv3`
- `character_book`
- `lore`

项目正在持续对齐 SillyTavern 最新源码中的角色卡 / 世界书结构与导出逻辑。

## 开发

```bash
npm install
npm run dev
```

构建：

```bash
npm run build
```

## 说明

- 当前是纯前端项目，AI 翻译请求由浏览器直接访问 OpenAI 兼容接口。
- 如果要做公网产品，建议后续补服务端代理、额度控制、审计与限流。
